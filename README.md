**Демо:** [![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/ваш-логин/ваш-репозиторий)

Веб-приложение для прогнозирования курса **рубля к доллару США (RUB/USD)** с использованием глубокого обучения (LSTM, LSTM+Attention, Transformer).  
Данные загружаются в реальном времени с сайта Центрального банка России, и вы можете дополнительно использовать **ключевую ставку ЦБ РФ** из локального файла `keyrate.txt`.  
Все прогнозы показываются в рублях за 1 доллар.

## ✨ Возможности

- **Получение данных**
  - Автоматическая загрузка дневных курсов USD и EUR с сайта ЦБ РФ (XML API)
  - Опциональная загрузка ключевой ставки из файла `keyrate.txt` (формат: `ДД.ММ.ГГГГ<разделитель>значение`)
  - Пропуски заполняются методом forward/backward fill для непрерывного временного ряда

- **Инженерия признаков**
  - Исходные курсы: USD, EUR, ключевая ставка
  - Технические индикаторы на основе USD:
    - SMA_7, SMA_30 (простые скользящие средние)
    - RSI (14‑дневный индекс относительной силы)
    - Волатильность (20‑дневное стандартное отклонение)

- **Архитектуры моделей** (выбираются в боковой панели)
  - Классическая LSTM (2 слоя)
  - LSTM с самовниманием (self‑attention)
  - Трансформер (Multi‑Head Attention + Feed‑Forward)

- **Настройка гиперпараметров**
  - Длина последовательности, горизонт прогноза, доля обучающей выборки
  - Количество нейронов, dropout, эпохи, batch size, learning rate

- **Обучение и оценка**
  - Интерактивный прогресс‑бар и графики потерь/MAE в реальном времени
  - Метрики: MAE, RMSE, R², MAPE
  - Модель можно скачать в формате HDF5

- **Прогнозирование**
  - Построение прогноза на свежих данных (последние доступные курсы)
  - Визуализация истории и прогноза
  - Таблица с дневными значениями (опционально обратный курс RUB/USD)
